# Hackathon-1-
El reto de este primer Hackathon consiste en crear una landing page interactiva que muestra los principales productos y servicios de una tienda deportiva.
La plataforma debera incluir lo siguiente:
1.Diseño atractivo
2.Interfaz intuitiva
3.Adaptarse a diferentes dispositivos
4.Aplicar animaciones, inserciones de productos al carrito,etc.
