# Hackathon-1: Landing Page Deportiva
## 📌 Descripción
El reto de este primer Hackathon consiste en desarrollar una landing page interactiva que muestre los principales productos y servicios de una tienda deportiva.
El objetivo es ofrecer una experiencia visual atractiva, intuitiva y adaptable a diferentes dispositivos, integrando animaciones y funcionalidades dinámicas como la inserción de productos al carrito.

## 🎯 Objetivos del Proyecto
Diseño atractivo: interfaz moderna con enfoque visual deportivo.

Interfaz intuitiva: navegación clara y accesible para cualquier usuario.

Responsive Design: adaptación a distintos dispositivos (desktop, tablet, móvil).

Interactividad: animaciones, transiciones y funcionalidades como agregar productos al carrito.

## 🛠️ Tecnologías Utilizadas
HTML5 → estructura semántica de la página.

CSS3 / Bootstrap → estilos, diseño responsivo y componentes visuales.

JavaScript (ES6) → interactividad, animaciones y lógica del carrito.

GitHub Pages → despliegue del proyecto.

## 🚀 Funcionalidades Principales
Catálogo de productos deportivos con imágenes y descripciones.

Botón de “Agregar al carrito” con actualización dinámica.

Animaciones en botones, secciones y transiciones.

Diseño adaptable a cualquier dispositivo.
